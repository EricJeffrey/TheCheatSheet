数据对象
    用户 = 标识 + 电子邮箱 + 名称 + 密码 + 赞过的代码段列表
    代码段 = 标识 + 标题 + 描述 + 内容 + 点赞数 + 标签列表

NoSQL数据库实体
    用户
        id              - 标识
        email           - 电子邮箱
        name            - 名称
        password        - 密码
        favors          - 赞过的代码段列表
    代码段 codeSegment
        id              - 标识
        title           - 标题
        description     - 描述
        content         - 内容
        thumbUpCount    - 点赞数
        tagList         - 标签列表
    标签
        id              - 标识
        value           - 标签值


功能列表
    自动登录
        登录成功设置cookie实现
        登录成功跳转到首页
    浏览查看 - 分页固定
        分页浏览所有代码
            按时间顺序
            按点赞数排序分页浏览
        分页浏览某个标签的代码
            按时间顺序
            按点赞数排序分页浏览
        查看某个代码段
        复制代码段内容
        点赞代码段
    搜索代码段
    编辑操作
        新增代码段
        修改某个代码段
    用户操作
        注册 - 完成后自动登入
        登入 - 设置cookie
        登出 - 清除cookie
        修改密码 - 重设cookie
        查看赞过的代码段


接口描述 - 与功能相对的前后端交互接口
    用户
        登录                                                - POST:login:{[name: xxx|email: xxx], password: xxx}
        注册                                                - POST:register:{email: xxx, name: xxx, password: xxx}
        更新密码                                            - POST:updatePassword:{email: xxx, oldPassword: xxx, newPassword: xxx}
        获取用户赞过的第n页数据                             - GET:userFavor?page=n
    浏览
        按[时间顺序|点赞数排序]获取[某个标签]的第n页数据    - GET:codeSegments?sortBy=[favorNumber|time]&[tag=xxx]
        获取所有标签                                        - GET:tagList
        赞某个代码段                                        - POST:favor{userId: xxx, codeSegmentId=xxx}
        获取第n页的搜索结果                                 - GET:search?text=xxx&page=n
    编辑
        更新某个代码段                                      - POST:updateCodeSegment:{codeSegmentId:xxx, {updatedData...}}
        新增代码段                                          - POST:addCodeSegment:{codeSegment}

杂项约束
    固定分页
    每个代码段对应多个标签
    默认时间顺序
    登录/登出后跳转到首页
    搜索结果不支持按标签分类


可扩展
    代码段被查看数
    创建时间/最后修改时间
    分享功能
    外部登录接口-微信/QQ...
    分类与标签
    标签使用词云
    用户头像
    绑定的社交账号
    用户点赞的代码段按标签

系统结构
    HTTP服务器 + MongoDB + Elasticsearch
